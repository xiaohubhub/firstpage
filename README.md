# VIB Coding 个人网页

这是一个赛博朋克风格的 VIB Coding 讲师个人网页，专为零基础学员提供编程教学服务。

## 项目特点

- 赛博朋克风格设计
- 玻璃拟态效果
- 响应式布局
- 流畅的交互动画

## 技术栈

- HTML5
- CSS3
- JavaScript

## 页面结构

1. 首页区域 - 个人简介和课程介绍
2. 教学特色 - 核心优势展示
3. 课程介绍 - 热门课程列表
4. 关于我 - 个人介绍和统计数据
5. 联系方式 - 社交媒体链接

## 如何使用

1. 克隆仓库：`git clone https://github.com/xiaohubhub/firstpage.git`
2. 打开 `index.html` 文件在浏览器中查看
3. 或直接访问：[https://xiaohubhub.github.io/firstpage/](https://xiaohubhub.github.io/firstpage/)
